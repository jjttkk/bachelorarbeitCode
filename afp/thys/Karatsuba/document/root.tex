\title{Karatsuba Multiplication for Integers}
\author{Jakob Schulz, Emin Karayel}
\maketitle

\begin{abstract}
We give a verified implementation of the Karatsuba Multiplication on Integers \cite{karatsuba} as well as verified runtime bounds.
Integers are represented as LSBF (least significant bit first) boolean lists, on which the algorithm
by Karatsuba \cite{karatsuba} is implemented. The running time of $O\left(n^{\log_2 3}\right)$ is verified
using the Time Monad defined in \cite{rbt}.
\end{abstract}
